# Student-Information-System
A console-based application developed in C++ that manages student records efficiently. The system allows users to perform basic CRUD operations (Create, Read, Update, Delete) on student data.
# Features
- Add new student records
- View existing records
- Update student information
- Delete student records
- File handling for data storage

# Technologies Used
- C++
- Dev-C++
- File Handling

# Learning Outcomes
- Understanding of file handling in C++
- Implementation of CRUD operations
- Improved problem-solving and logic building
